Cursor

The AI-first Code Editor for massive productivity gains.

Built on top of VS Code, Cursor is an AI-native code editor designed to help you write code faster. It integrates large language models directly into the IDE environment, enabling advanced features like codebase-wide indexing and smart refactoring.

Technical Capabilities

Cursor enables you to interact with your codebase using natural language. It indexes your entire repository, allowing the AI to understand relationships between files, suggest complex refactors, and predict your next move with "Cursor Tab" (Copilot++).

Unlike standard extensions, Cursor is a fork of VS Code, meaning the integration is deeper and more performant. It supports "Composer" mode for multi-file edits and provides one-click terminal commands.
